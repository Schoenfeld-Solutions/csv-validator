# DATEV CSV file validator

Independent open-source tool. Not affiliated with DATEV eG.

This project provides a public, browser-only validator for DATEV-like CSV files.

## Key guarantees

- Runs fully client-side in the browser.
- No file upload.
- Deterministic and fail-closed validation.
- Contract-driven structural and semantic checks.

## Warranty disclaimer

Provided **as-is**, without guarantees and without warranty of any kind.

## Compliance boundary

- No code or assets from official DATEV sources.
- No DATEV logos or official brand assets.
- DATEV name is used descriptively only.
